# Проектный практикум — АИС «Контроль текущей успеваемости студентов»
## Автоматизированная информационная система учёта академической успеваемости

---

## Тема проекта

**Название**: АИС «Контроль текущей успеваемости студентов» — веб-приложение для регистрации, хранения и анализа результатов учебной деятельности студентов.

**Суть**: Преподаватели выставляют оценки за контрольные работы, лабораторные, экзамены и другие виды учебных мероприятий. Студенты видят свои оценки и историю изменений в личном кабинете. Деканат получает сводную аналитику успеваемости в режиме реального времени.

---

## Стек технологий

| Слой | Технология |
|------|-----------|
| Frontend | Nuxt 3 (SPA, SSR отключён) + TypeScript |
| UI-библиотека | Naive UI |
| Стейт-менеджмент | Pinia 2 |
| HTTP-клиент | Axios |
| Стили | SCSS (адаптивный, mobile-first) |
| Backend | Go 1.21+ + Gin v1.9 |
| БД-драйвер | `database/sql` + `lib/pq` (raw SQL, без ORM) |
| СУБД | PostgreSQL 15+ |
| Аутентификация | JWT HS256 (Access 15 мин + Refresh 30 дней) |
| API | REST JSON (`/api/v1/`) |
| E2E-тесты | Playwright (desktop + mobile) |

## Роли пользователей

| Роль | Значение в БД | Описание |
|------|--------------|---------|
| Студент | `student` | Просматривает свои оценки, посещаемость, историю изменений |
| Преподаватель | `teacher` | Выставляет и редактирует оценки, добавляет комментарии |
| Администратор (деканат) | `admin` | Управляет дисциплинами, пользователями, просматривает аналитику |

## Основные функции системы

### Для студента (`student`)
- Регистрация / вход по email и паролю
- Просмотр списка своих оценок с фильтрацией по дисциплине, типу оценки, статусу
- Просмотр детальной карточки оценки: балл, комментарии преподавателя, история изменений
- Подтверждение ознакомления с оценкой

### Для преподавателя (`teacher`)
- Просмотр очереди на оценивание (работы без выставленного балла)
- Создание записи об оценке: студент, дисциплина, тип работы, название, балл
- Изменение балла с обязательным комментарием-обоснованием (фиксируется в историю)
- Смена статуса: `Ожидает → Оценено → Аттестован / Не аттестован`
- Добавление публичных комментариев и внутренних заметок (скрытых от студента)

### Для администратора (`admin`)
- Управление пользователями: смена роли, блокировка/разблокировка
- Управление дисциплинами: создание, редактирование, удаление
- Управление типами оценок: Текущая / Рубежная / Итоговая + максимальный балл
- Просмотр дашборда: всего оценок, ожидают оценки, средний балл, распределение по статусам, топ преподавателей
