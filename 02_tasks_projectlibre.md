## Как заполнить ProjectLibre

1. **Файл → Новый** → указать название «АС ServiceDesk», дату начала 02.04.2025
2. **Вкладка «Ресурс»** → добавить 3 ресурса из таблицы выше
3. **Вкладка «Задача»** → для каждой строки:
   - Двойной клик → заполнить «Название», «Продолжительность», даты
   - Вкладка «Ресурсы» → назначить исполнителя
   - Вкладка «Предшествующие» → указать номер предшествующей задачи
4. **Сохранить** → диаграмма Ганта построится автоматически
5. **Задача → Сеть** → просмотр сетевой диаграммы зависимостей
